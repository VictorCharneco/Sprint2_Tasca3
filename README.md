# Sprint2_Tasca3 : Òptica – Cul d’Ampolla (MongoDB)
## Què és?

Això és un exercici per modelar una base de dades en MongoDB (no SQL) per a una òptica.


#### Exercici 1: 
vista del client (fitxa + últimes compres)

#### Exercici 2: 
vista de les ulleres (fitxa + comprat per + proveïdor)

## Tecnologies

MongoDB

MongoDB Compass

## Fitxers

Nivell_1/exercici_1_clients.json → col·lecció clients

Nivell_1/exercici_2_ulleres.json → col·lecció ulleres

## Com provar-ho (Compass)
1. Crear la base de dades: `cul_dampolla`
2. Importar `exercici_1_clients.json` a la col·lecció `clients`
3. Importar `exercici_2_ulleres.json` a la col·lecció `ulleres`
